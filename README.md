## Team Members

- Princess Krish Anne Bernardino

## Project Description

Stock Society is an inventory management web application designed for a clothes warehouse. The system allows warehouse staff to manage products, inventory, suppliers, distribution centres, orders, customers, and shipments.

## Technologies

- Spring Boot
- Thymeleaf
- Spring Security
- Spring Data JPA
- Spring Data REST

## How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run the Spring Boot application
4. Open localhost:8080
